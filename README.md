# AMusicPlayer (Mac 优化版)

An English introduction: This is a macOS-optimized edition of AlgerMusicPlayer. It keeps the original features intact, with small UI refinements to better fit the macOS experience.

本项目基于原作者的项目进行优化与调整，谨此致敬并感谢原作者的辛勤付出。

原作者项目地址：
https://github.com/algerkong/AlgerMusicPlayer

## 说明

本项目主要针对 macOS 进行优化，重点做了部分界面调整，并移除了一些与功能无关的内容。

## 主要改动

1. 图标修改（适配 macOS Menu Bar）
2. 配色轻微调整（浅绿色 → 浅蓝色，未修改进度条）
3. 界面仅保留基础功能，删除多余元素（对不起原作者的大大，求原谅 QaQ）
4. 适配 macOS 关闭与最小化图标
5. 在此基础上重新调整各布局位置

由于我的水平有限，不期望合并 PR。原作者的大版本可能会跟随更新，小版本暂时不跟；大 Bug 发邮件，小 Bug 提 Issue，随缘修复。

其中移除的内容主要包括原作者的赞助码。我非常尊重原作者的劳动与付出，因此在本项目中移除赞助码是为了避免误导用户。如果你认可原作者的努力，请前往原作者项目进行支持与赞赏，我本人不接受任何赞助。
